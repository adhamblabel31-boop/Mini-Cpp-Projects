## 🚀 Build & Run

Ensure you have the GNU Compiler Collection (`g++`) installed.

### 1. Clone the Repository

git clone [https://github.com/adham/cpp-utilities.git](https://github.com/adham/cpp-utilities.git)
cd cpp-utilities

### 2. Compile and Run

Each file is standalone. Use the following commands to execute a specific module:

**Gym System**

g++ gym.c++ -o gym
./gym

**Salary Calculator**

g++ salary.c++ -o salary
./salary

**Grading System**

g++ school.c++ -o school
./school
